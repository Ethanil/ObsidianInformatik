# Assembler-Befehle 
`CC` steht für [[Condition Codes]] und S für mögliche Suffixe um [[Statusflags]] zu setzen.
## mögliche Operanden
- Direkter Wert (I)
- Register (R)
- Speicher (M)
- Adresse (PTR)
## Arithmetische Befehle
| Befehl | Operanden | Statusbits | Beispiel |
| ------ | --------- | ---------- | -------- |
| Add    | R,R,R/I   |  N,C,Z,V          |          |

# [[Operatoren#Modulo|Modulo]] Rechenregeln
Bei Rechnungen mit Modulo kann vieles vereinfacht werden, wenn man Rechenregeln anwendet:

$$(a + b) \ mod \ c = ((a \ mod \ c) + (b \ mod \ c)) \ mod \ c$$
$$(a * b) \ mod \ c = ((a \ mod \ c) * (b \ mod \ c)) \ mod \ c$$
$$(a^b) \ mod \ c = ((a \ mod \ c) ^ b) \ mod \ c$$
